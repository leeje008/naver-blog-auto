## 1. 네이버 검색 알고리즘 이해

### C-Rank (Creator Rank) — 블로그 전체 신뢰도

| 요소 | 비중 | 설명 |
|------|------|------|
| 주제 전문성 (Context) | ~40% | 특정 분야에 집중한 콘텐츠 |
| 활동 지속성 (Content) | ~30% | 정기적이고 일관된 포스팅 |
| 사용자 반응 (Chain) | ~20% | 댓글, 공유, 상호작용 |
| 콘텐츠 인기도 | ~10% | 조회수, 방문자수 |

### D.I.A.+ (Deep Intent Analysis Plus) — 개별 문서 품질

1. **문서의 주제 적합도** — 검색어와 문서 내용의 관련성
2. **경험 정보** — 실제 체험기, 후기, 사용기
3. **정보의 충실성** — 상세하고 깊이 있는 정보
4. **문서의 의도** — 순수 정보 제공 vs 광고/홍보
5. **상대적 어뷰징 척도** — 스팸 여부
6. **독창성** — 중복/복사 콘텐츠 탐지
7. **적시성** — 최신 정보 여부

### 2025-2026 트렌드

- 시맨틱 검색 강화: 키워드 매칭보다 의도 일치가 더 중요
- AI 생성 콘텐츠 감시 시스템 도입
- 모바일 우선 인덱싱 (2024 Q4~)

## 2. SEO 최적화 핵심 파라미터

### 2.1 제목 (Title)

| 항목 | 권장값 |
|------|--------|
| 길이 | 25자 이내 (한글) |
| 타이틀 태그 | 40자 이내 (SERP 표시) |
| 키워드 위치 | 제목 시작 부분 |
| 형식 | 구체적 숫자 + 형용사 활용 |

### 2.2 본문 구조

```
[권장 구조]
1. 도입부 (100~200자) — 키워드 필수 포함, 글 목적 설명
2. 본론 (1,500~2,000자) — H2/H3 소제목, 이미지 배치
3. 결론 (200~300자) — 핵심 정리, 키워드 재언급, CTA
```

| 항목 | 권장값 |
|------|--------|
| 전체 본문 | 2,500~3,000자 (최소 1,500자) |
| 한 단락 | 300~500자 |
| 한 문장 | 30~50자 |
| 최대 한도 | 4,000~4,500자 |

### 2.3 키워드 밀도

| 항목 | 권장값 |
|------|--------|
| 전체 밀도 | 1~2% (과도 시 스팸 판정) |
| 핵심 키워드 횟수 | 1,500~2,000자 기준 5~6회 |
| 제목+본문 최소 | 3~5회 |

**위치별 배치 규칙:**
1. 제목: 시작 부분에 1회
2. 초반 200자: 1~2회 (필수 — 알고리즘 최우선 분석 구간)
3. 소제목(H2/H3): 키워드 변형 포함
4. 본문 중간: 자연스럽게 2~3회 분산
5. 마지막 200자: 1회 재언급
6. 이미지 ALT: 키워드 포함
7. 해시태그: 핵심 키워드 포함

### 2.4 HTML Heading 구조

- H1: 페이지당 정확히 1개
- H2: 1,000자당 3~5개 (체류 시간 최대 50% 증가)
- 계층 순서 유지 (H1 → H2 → H3, 건너뛰기 금지)
- H1, H2에 키워드 자연 포함
- 본문은 `<p>` 태그 사용 (`<div>`, `<span>` 지양)

### 2.5 이미지 SEO

| 항목 | 권장값 |
|------|--------|
| 해상도 | 800~1,200px (가로) |
| 파일 크기 | 압축 후 100~300KB |
| 개수 (정보성) | 6~13장 |
| 개수 (리뷰) | 20~30장 |
| 개수 (최소) | 3장 이상 |
| ALT 텍스트 길이 | 20~50자 (한국어, 문장형) |

**ALT 태그 규칙:**
- 좋은 예: `alt="강남역 근처 이탈리안 레스토랑의 트러플 파스타"`
- 나쁜 예: `alt="맛집 파스타 강남 레스토랑 추천 맛있는 음식"`

**이미지 배치 전략:**
1. 첫 이미지: 본문 시작 직후 (대표 이미지)
2. 각 H2 소제목 아래: 관련 이미지 1~2장
3. 300~500자마다 이미지 삽입
4. 유사/중복 이미지 연속 금지

### 2.6 해시태그

| 항목 | 권장값 |
|------|--------|
| 개수 | 5~10개 (핵심만) |
| 최대 | 30개 |
| 필수 포함 | 핵심 키워드 + 지역 키워드 |

### 2.7 AI 콘텐츠 안전 가이드라인

1. AI 생성 후 반드시 인간 편집 거치기
2. 개인 경험/인사이트 최소 20~30% 추가
3. 반복적 문구 패턴 제거
4. 자연스러운 구어체 혼합
5. 구체적 사례와 수치 데이터 포함
6. 출처/근거 링크 추가
7. 동일 패턴의 글 연속 게시 금지

## 3. 저품질 방지 체크리스트

### 절대 금지 사항
- 키워드 과다 삽입 (stuffing)
- 광고/홍보성 문장 과다
- 중복 콘텐츠 반복 게시
- AI 생성 콘텐츠 수정 없이 대량 게시
- 유사 이미지 반복 사용
- 잦은 게시글 수정 (노출에 악영향)
- 클릭베이트 제목

### 필수 실행 사항
- 고유한 경험/인사이트 포함
- 신뢰할 수 있는 출처 명시
- 정보성 글 비율 높이기
- 발행 리듬 안정적으로 유지
